Statistics Fundamentals

This section describes various statistical mathematical concepts, that are implemented in the "Hello Stats" app. It is intended to give a quick overview of statistics theory, to users who would like to refresh these concepts before making use of this app for any serious use.

(1) Average or Measures of Central Tendency

a) Arithmetic Mean

If x1,x2,x3, ...... xn are n numbers, then their arithmetic mean is defined by:

(x1 + x2 + x3 + ..... + xn) / n

b) Geometric Mean

If x1,x2,x3, ...... xn are n numbers, then their geometric mean is defined by:

pow ((x1 * x2 * x3 ...... ), 1/n)    i.e the result of multiplying all the numbers, and then finding the nth root of it. In this formula, "pow" refers to a power value calculation.

c) Harmonic Mean

If x1,x2,x3, ...... xn are n numbers, then their harmonic mean H is defined by:

1/H = (1/n) * (1/x1 + 1/x2 + 1/x3 ...... 1/xn)

d) Median

Let there be n numbers. To find the median of these given set of numbers, the following procedure is followed:

- Arrange the numbers in ascending or descending order of magnitude.

- When the total number of the items is odd and equal to say n, then the value of (1/2) * (n + 1) th item gives the median.

- When the total number of the items is even and equal to say n, then there are two middle items, and the mean of the values of n/2 th and (n/2 + 1) th items is the median.

e) Mode

Mode is that value in sample data which occurs most frequently. It is possible to have more than one such value in the sample.

f) Average Deviation

This is defined as the mean of the absolute values of the deviations of a given set of numbers from their arithmetic mean.

Let x1,x2,x3 ..... xn be a set of numbers with frequencies f1,f2,f3 ..... fn respectively. If x is the arithmetic mean of the numbers x1,x2,x3 .... xn   then, Average deviation is defined as follows:

(f1* |x1 - x| + f2* |x2 - x| + ....... fn* |xn - x|) / (f1 + f2 + ...... fn)

It is also correct to think of this calculation, without considering the concept of the "frequency" of occurrence of values. If x1,x2,x3 ..... xn are a set of numbers (a specific number can occur any number of times in such a set. the total number of occurrences of a number, is actually the frequency of that number) with x as arithmetic mean, then the Average deviation of these set of numbers is defined as follows:

(|x1 - x| + |x2 - x| + ...... |xn - x|) / (x1 + x2 + ..... + xn)

g) Standard Deviation

The standard deviation of a set of values x1,x2,x3 ..... xn is defined as follows:

Let m be the arithmetic mean of these set of numbers. The variance of these numbers is defined as follows:

VAR. = (1/n) * [(x1 - m)*(x1 - m) + (x2 - m)*(x2 - m) + ....... (xn - m)*(xn - m)]

Statistics theorists have different opinions about, that what size of the data list must be considered when using this formula to calculate variance and therefore standard deviation. Popular beliefs are, that in the above formula we can either have 1 / (n - 1) or 1 / n and the rest of formula remains same.

The standard deviation (SD) is square root of VAR.  i.e  SD = sqrt(VAR.)

(2) Skewness

Skewness denotes the opposite of symmetry. In a symmetrical series the arithmetic mean, the median and the mode are identical. Skewness is defined by the following formula:

(Mean - Mode) / (standard deviation)

(3) Correlation and regression

Whenever two variables are related in a way, that an increase in value of the one causes an increase or decrease in the value of other, then the variables are said to be correlated.

There are various methods in statistics for finding the correlation between two variables. The Hello Stats app implements the Pearson's coefficient of correlation, which is defined as follows:

Let m1 and m2 be arithmetic means of two sets of numbers x1,x2,x3 .....  and  y1,y2,y3 ..... respectively. Then coefficient of correlation is defined as following:

Let P = (x1 - m1)*(y1 - m2) + (x2 - m1)*(y2 - m2) + ...... + (xn - m1)*(yn - m2)

F = (x1 - m1)*(x1 - m1) + (x2 - m1)*(x2 - m1) + ...... (xn - m1)*(xn - m1)

G = (y1 - m2)*(y1 - m2) + (y2 - m2)*(y2 - m2) + ...... (yn - m2)*(yn - m2)

Coefficient of correlation = P / sqrt(F * G)

A related concept with coefficient of correlation is, equation of a regression line. There is a type of regression line (a straight line on an x-y axis) termed as "least-squares regression line", whose equation is defined as follows:

y = a + bx

where b = r * (Sy/Sx)

a = Ym - b*Xm

Where r = coefficient of correlation, Sx = standard deviation of X values, Sy = standard deviation of Y values, Xm = arithmetic mean of X values, Ym = arithmetic mean of Y values.
